# Fahad Jewellery Box

A responsive static business website for **Fahad Jewellery Box**, a jewellery box manufacturing business with over 20 years of experience in the industry. The site showcases the company's products, craftsmanship, and story, and provides visitors with an easy way to get in touch.

## Description

This project is the official website for Fahad Jewellery Box. It presents the company's background, product range, and reasons customers should choose them, along with a gallery of work and a contact section. The site is built as a static, framework-free web application optimized for performance and responsiveness across devices.

## Technologies Used

- **HTML5** — semantic markup and page structure
- **CSS3** — styling and responsive layout (no CSS frameworks)
- **JavaScript (Vanilla)** — client-side interactivity (no JS libraries/frameworks)
- **Git & GitHub** — version control
- **Vercel** — deployment/hosting
